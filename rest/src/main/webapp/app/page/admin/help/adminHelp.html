Administration Help

Documentation on the "Admin" tab of the IHTSDO Refset/Translation tool.

Validation Checks

Information on validation checks.

  • Default validation check:
    • For refsets: require name, description, and module id. Ensure only INTENSIONAL has a definition, only EXTERNAL has an external URL, and EXTERNAL does not have any members
    • For refset members: INCLUSION or EXCLUSION are only associated with INTENSIONAL refsets
    • For translations: require language to be set. Warn if language is > 2 characters
    • For concepts: warn non descriptions with trailing, leading, or duplicate whitespace. Require length to be within description type limits.
